# Dot Files for [brijr](https://bridger.to)

1. Clone the repo:

```bash
git clone https:://github.com/brijr/dot.git
```

2. Change into the directory:

```bash
cd dot
```

3. Make it executable:

```bash
chmod +x setup.sh
```

4. Run it:

```bash
./setup.sh
```

If you get a permission error, try:

```bash
sudo ./setup.sh
```
